# PycMan

A recreation/re-make of the classic arcade game Pac-Man built using Python and Pygame.

This project is part of a collection of game recreation made by me in order to learn about game development, game programming and game design.
Every mechanic was implemented from scratch.

## Features

- A* pathfinding algorithm implementation.
- Matrix-based map generation.
- Code-driven animations and sound management.
- Main menu and gameplay screen.
- Randomized ghost behavior.

## What I Learned and Improved

- Object-Oriented Programming (OOP)
- Pathfinding algorithms (A*)
- Game loops and state management
- Collision detection
- Sprite animation systems

## Technologies

- Python 3
- Pygame 2

## Controls

- WASD or arrow keys to move
- Space or Enter key to start game

## Instalation
```bash
git clone https://github.com/lautttdev/PythonInvaders.git
cd PythonInvaders
pip install pygame
python main.py
```
